# SkyForce - Java Shooting Game

SkyForce is a simple 2D arcade-style shooting game developed using Java. In this game, the player controls an aircraft that moves forward, shoots obstacles or enemies, and gains points by destroying them. The main goal is to survive as long as possible, avoid collisions, and achieve the highest score.

## Features

- Player-controlled aircraft
- Shooting system to destroy obstacles/enemies
- Score system based on successful hits
- Increasing challenge as the game progresses
- Collision detection between bullets, obstacles, and the aircraft
- Simple and interactive gameplay
- Java-based graphical user interface

## Gameplay

The player controls a plane that moves through the game screen. Obstacles or enemies appear in front of the plane, and the player must shoot them to gain points. If the plane collides with an obstacle, the game may end or the player may lose health, depending on the game logic.

The objective is to keep moving forward, destroy as many obstacles as possible, and collect the highest score.

## Controls

| Key | Action |
|---|---|
| Arrow Up | Move Up |

| Spacebar | Shoot |

## How to Run the Game

1. Clone or download this repository.

```bash
git clone https://github.com/your-username/skyforce-java-game.git
```

2. Open the project in any Java-supported IDE, such as:

- IntelliJ IDEA
- Eclipse
- NetBeans
- VS Code

3. Make sure Java JDK is installed on your system.

4. Compile and run the main Java file.

```bash
javac Main.java
java Main
```
